# AI-Powered-Customer-Review-Sentiment-Analysis-BI-Chatbot
An end-to-end Python project that performs sentiment analysis on customer reviews using NLP, generates business insights, visualizes trends, and includes an interactive rule-based BI chatbot.
This project is a complete Customer Review Analytics System built using Python and NLP techniques.
It analyzes thousands of customer reviews, predicts sentiment (Positive, Negative, Neutral), generates business insights, creates visual dashboards, and allows users to interact with the system through a simple chatbot.

The project demonstrates:
Natural Language Processing (NLP)
Sentiment Analysis
Business Intelligence
Data Visualization
Interactive Chatbot Development

Sentiment Analysis
Uses TextBlob NLP for sentiment prediction
Classifies reviews into:
Positive
Negative
Neutral
Generates polarity scores
✅ Business Intelligence Insights
Finds:
Most negative product category
Worst platform based on reviews
Best-performing month
Top customer complaints
✅ Data Visualization

Automatically generates charts:

Sentiment Distribution Pie Chart
Category-wise Sentiment Bar Chart
Monthly Trend Analysis
Yearly Sentiment Trends
✅ Interactive BI Chatbot

Users can ask questions like:

“What is the overall sentiment?”
“What are the top complaints?”
“Show sentiment trends”
“Which platform has most negative reviews?”
